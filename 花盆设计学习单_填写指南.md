## 第一部分：科学探究

### 探究1：陶泥选择

**我们的选择**: 紫砂泥或粗陶土（已填）

**理由**（需填写）:
```
百里香原产地中海，喜干燥排水良好的环境。紫砂泥和粗陶土具有较高的吸水率和透气性，能快速排出多余水分，防止根部积水腐烂，同时保持适当的湿度，非常适合百里香的生长习性。
```

### 探究2：烧成温度

**吸水性选择**: 偏高（需圈选）

**烧成温度范围**: 850℃ ～ 1050℃（已填）

**理由**（需填写）:
```
百里香需要良好的排水透气性，850-1050℃的中低温烧制可以使陶土保持一定的孔隙率，吸水率在5-10%之间，既能快速排水又能保持一定湿度，避免根部过干或过湿。
```

### 探究3：盆的尺寸

**植物尺寸**: 冠幅约15cm，高度约12cm（已填）

**设计的盆器尺寸**（已填）:
- 盆口内径：12-15 cm
- 盆高：10-12 cm
- 盆底内径：8-10 cm
- 排水孔：1个，直径约8-10 mm

**设计依据**（需填写）:
```
百里香根系较浅但发达，主根深度约8-10cm，侧根横向扩展。盆口直径12-15cm（约为冠幅的0.8-1倍）可提供足够的生长空间，盆高10-12cm与根系深度匹配，盆底内径8-10cm形成适度锥度利于排水，排水孔8-10mm确保排水通畅。
```

## 第四部分：陶艺技法

### 技法选择

**我选择的技法**（需填写）:
```
泥条盘筑 + 手捏成型
```

**选择理由**（需填写）:
```
百里香花盆采用圆柱形造型，泥条盘筑适合制作开口较宽的圆形盆器，可以控制盆壁厚度和曲线变化。手捏成型用于细节处理，如盆口边缘的圆润过渡和排水孔的制作。两种技法组合，既保证结构稳定性，又能体现手工质感。
```
