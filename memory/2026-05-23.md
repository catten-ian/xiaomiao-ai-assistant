# 2026-05-23 Charade 2 增强功能开发

### 阶段1: 数据库变更 ✅
- rooms 表新增：password, difficulty, timeout_seconds, categories 字段
- players 表新增：is_ai 字段
- 新增 game_stats, player_stats, round_stats 表
- 词库从 68 → 947 个词（12个类别，easy/medium/hard 三级）

### 阶段2: 房间密码保护 ✅

- 支持可配置超时时间（30-120秒）
- 描述者和猜测者都有超时检测
- 前端倒计时准备中

### 阶段4: AI 玩家系统 ✅
- AI 玩家 API (ai_player.php)
- 集成 v3cm API (DeepSeek-V4-Flash)
- AI 描述测试：熊猫→"黑白毛色圆滚滚" ✅
- AI 猜测测试：猜中"熊猫" ✅
- 支持多AI玩家（小智、小灵、阿杰、梦瑶）

### 积分系统更新 ✅
- 按难度计分：easy=5, medium=10, hard=15
- 速度奖励：10秒内猜对+5分
- 描述者和猜测者都获得积分

### 游戏流程增强 ✅

- 难度渐进：1-2轮easy, 3-4轮medium, 5轮hard
